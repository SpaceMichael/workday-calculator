# 返工日數計算器

這是一個純前端 HTML 網站，用來計算員工返工的**純日曆天數**。

## 功能

- 輸入開始上班日與最後上班日
- 日期格式使用 `dd/mm/yyyy`
- 計算結果**包含首尾兩日**
- 自動處理閏年
- 不扣香港公眾假期
- 日期無效或結束日早過開始日時，會顯示錯誤

## 使用方法

1. 用瀏覽器開啟 `index.html`
2. 輸入開始上班日
3. 輸入最後上班日
4. 按「計算日數」

## 檔案

- `index.html`：主頁面與計算邏輯

## 顯示結果

畫面會顯示：

- 總日數
- 首尾日期說明
- 錯誤訊息（如日期格式不正確）

## 範例

- 開始：`01/04/2026`
- 結束：`01/04/2026`
- 結果：`1` 日
